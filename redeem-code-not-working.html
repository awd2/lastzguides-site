Last Z Code Not Working? Gift Center, UID, and Reward Fixes

Troubleshooting • Updated March 2026 • Official Gift Center flow • Last Z Guides

Quick Answer

Why your Last Z code is not working: most failures come from a wrong UID, typo, expired code, already-used code, or checking the wrong place for rewards after redeeming through the Gift Center.

🌐 Use the official page: codes are redeemed through the Gift Center, not the game client.

🆔 Copy UID directly: Avatar → Settings → Copy ID is safer than typing it manually.

📬 Check your mailbox: rewards go to in-game mail, not the Gift Center screen.

i Sort the failure first: wrong UID or typo means the redemption failed, expired or already-used means the code is no longer claimable for that account, and missing rewards means you should check mailbox timing before retrying.

First 5 Checks When a Last Z Code Does Not Work

  1. Make sure you are using the official Gift Center.
  2. Copy your UID directly from Settings instead of typing it manually.
  3. Paste the code exactly as shown, with correct letter case.
  4. Check whether the code was already redeemed on this account.
  5. Open your in-game mailbox after redeeming and wait a few minutes if needed.

Need clean setup first? Use the Gift Center & UID Guide. Need current active codes? Use Redeem Codes.

Wrong UID Is One of the Most Common Problems

If the UID is wrong, the code will fail even when the code itself is still active. This is one of the fastest things to check.

  1. Open Last Z
  2. Tap your Avatar in the top-left corner
  3. Open Settings
  4. Tap Copy ID
  5. Paste that UID into the Gift Center

If you need the clean setup flow for Gift Center login and UID copying instead of troubleshooting, use the Gift Center & UID Guide.

For the full redemption walkthrough and the current verified codes list, use the Redeem Codes page.

Expired and Already-Used Codes

There is an important difference between an expired code and a code that was already used on your account.

  • Expired: the code no longer works for anyone.
  • Already used: the code may still be valid in general, but not for your account anymore.
  • Unverified repost: the code may have been shared online before official confirmation.

Rewards Not Showing Up? Check Your Mailbox

Rewards are delivered to your in-game mailbox, not shown instantly on the Gift Center page. If nothing appears right away, wait a few minutes and restart the game once before trying again.

Can You Redeem Last Z Codes Inside the Game?

No. Last Z codes are redeemed through the official Gift Center website, not inside the game app. This is true on both iPhone and Android.

Does the Same Fix Work on iPhone and Android?

Yes. The redemption flow is the same on iPhone and Android because both use the official Gift Center in a browser.

  • iPhone: open the Gift Center in Safari or another browser, then paste your UID and code.
  • Android: open the Gift Center in Chrome or another browser, then paste your UID and code.
  • In-game: not supported for code redemption.

When the Problem Is Not on Your Side

Sometimes the Gift Center page itself is slow, temporarily unavailable, or delayed after a fresh code announcement. If you already checked UID, code spelling, and mailbox delivery, the issue may simply be temporary site or server lag.

  • Gift Center page fails to load
  • Code was announced very recently and redemption is delayed
  • Rewards are slow to appear in the mailbox

FAQ

Why is my Last Z code not working?

The most common reasons are a wrong UID, expired code, already-used code, typo, redeeming in the wrong place, or checking the wrong place for rewards.

Can I redeem Last Z codes inside the game?

No. Last Z codes are redeemed through the official Gift Center website, not inside the game client.

How do I find my UID in Last Z?

Open Avatar, go to Settings, and tap Copy ID. Paste that UID into the Gift Center instead of typing it manually.

Why are my rewards not showing up?

Rewards are sent to your in-game mailbox. Wait a few minutes, then restart the game once and check your mail again.

Can I redeem codes on iPhone or Android?

Yes. On both iPhone and Android, redemption happens through the official Gift Center website in your browser.

What if the Gift Center page does not load?

If the official Gift Center page fails to load, wait and try again later. Sometimes the problem is temporary site availability rather than your code.

Before Retrying A Failed Code

  • Checked in game with server 403 players: most failed redeems still come from expired codes, already-used codes, wrong UID, copied text mistakes, or Gift Center delay.
  • Before retrying, use the official Gift Center, copy your UID from the game, match the code text exactly, and check mailbox after a successful redeem.
  • If the Gift Center is down or the error text changes, wait and follow the live Gift Center response first.
