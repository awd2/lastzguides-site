Last Z Event Center Guide - What to Check Before Events

Event support guide • Updated June 19, 2026 • Last Z Guides

Quick Answer

Event Center is the in-game event screen in the top-right corner of Last Z. Use it to manage event-specific settings, check event progress, view reward screens, and open special event tasks.

  1. 1 Open it top-right - the current UI label is Event Center.
  2. 2 Keep Engage Offline enabled - it protects Tyrant rewards if you are offline.
  3. 3 Use progress tabs - Furylord ranking, Danger Lurks Boomers, and Territory Triumph Cities live there.
  4. 4 Claim extra rewards - Doomsday Club and Follow Our Community can lead to tasks or Diamonds.

What the Event Center Is For

Event Center is not a strategy page by itself. It is the live in-game screen where Last Z shows current and upcoming event details.

Use the Events Schedule for the broad weekly rotation. Use Event Center when you need the live event screen: registration stages, reward screens, rankings, event progress, or event-specific settings.

The value of this page is knowing what each Event Center entry is actually useful for, so you do not miss settings, progress checks, or extra rewards hidden behind event tabs.

What to Check Before an Event Starts

Before a scheduled alliance event starts, open Event Center once and check the live state:

  1. Whether the event is in preview, registration, matchmaking, combat, reward, or finished state.
  2. Any alliance setup requirement, especially for R4/R5-managed events.
  3. Any event-specific checkbox or setting, such as Engage Offline for Tyrant.

After that, most repeated value comes from using the right tab for the right job: rankings, Boomers, captured Cities, auto-join, website tasks, or social rewards.

Tyrant Event Center Check

For Tyrant, open Event Center -> The Tyrant before the event starts.

The important setting is Engage Offline. Keep this checkbox enabled so your troops can still join if you are offline during Tyrant. It is a small check, but missing it can cost easy rewards when the event starts while you are away.

Canyon Clash Event Center Check

Canyon Clash has several visible stages in Event Center: voting, registration, matchmaking, combat, and rewards.

This is mainly useful for confirming whether your alliance is still setting up, already matched, in combat, or ready to claim rewards.

For strategy after you enter the event, use the Canyon Clash guide.

Zombie Siege Event Center Check

For Zombie Siege, R4/R5 can schedule the start time from Event Center.

The best default is to schedule Zombie Siege on Monday when possible, because it lines up better with Alliance Duel day planning. If your alliance schedules it later, follow the live Event Center time and alliance instructions.

Progress and Extra Rewards

Several Event Center entries are useful even when you already know the event timing.

Furylord is where you can check your ranking. Danger Lurks is where you can track how many Boomers were killed and control Auto-join for alliance teammate rallies. Territory Triumph shows the currently captured Cities and the rewards for capturing them.

Two entries are more about extra reward tasks than event strategy. Doomsday Club sends you to the game website to complete tasks and claim additional rewards. Follow Our Community can give Diamonds for following the game's social channels.

Rewards and Missed Claims

If you do not claim rewards manually in Event Center, unclaimed rewards are sent by in-game mail.

That means missing the claim button is usually not fatal. Manual claiming is still cleaner because you can see the event result and reward screen before moving on.

Common Mistakes

  • Forgetting the current event stage: Canyon Clash and other alliance events can move through setup, combat, and rewards.
  • Leaving Engage Offline unchecked for Tyrant: keep it enabled as a fallback if you cannot join live.
  • Missing Zombie Siege scheduling: R4/R5 can set the start time, and Monday is usually the cleaner Alliance Duel fit.
  • Ignoring progress tabs: Furylord ranking, Danger Lurks Boomers, and Territory Triumph Cities are worth checking.
  • Forgetting extra reward entries: Doomsday Club and Follow Our Community can hide simple rewards outside normal event strategy.

Treat Event Center as a small control panel, not just a calendar.

FAQ

Where is Event Center in Last Z?

Event Center is in the top-right corner of the main game screen. The current UI label is Event Center.

Is Event Center the same as the event schedule?

No. The event schedule explains the usual rotation. Event Center is where you manage event settings, stages, progress checks, rankings, and extra reward tasks.

What should I check in Event Center before Tyrant?

Open The Tyrant entry and keep Engage Offline enabled so your troops can still join if you are offline.

What Canyon Clash stages are visible in Event Center?

Canyon Clash can show voting, registration, matchmaking, combat, and rewards.

Who can schedule Zombie Siege?

R4/R5 can schedule the Zombie Siege start time from Event Center. Monday is preferred when possible because it fits Alliance Duel planning better.

What happens if I forget to claim Event Center rewards?

Unclaimed rewards are sent by in-game mail, but manual claiming is cleaner because you can see the reward state and event result.
