Puis-je utiliser les codes Last Z sur iPhone ou Android ?

Oui. Le flux de redemption est le même sur iPhone et Android, car les codes sont réclamés via le site officiel du Gift Center, pas dans l’application du jeu.

Comment trouver votre UID dans Last Z

Votre UID est requis pour chaque redemption de code. Si l’UID est incorrect, le code échouera même si le code lui-même est encore actif.

Avant d’utiliser un code, vous avez besoin de votre UID (User ID) :

  1. Ouvrez Last Z: Survival Shooter
  2. Touchez votre Avatar (coin supérieur gauche)
  3. Allez dans Settings
  4. Touchez Copy ID
  5. Votre UID est maintenant copié dans le presse-papiers

Si vous voulez la version configuration de ce flux sans la liste complète des codes, utilisez le guide Gift Center & UID.

Comment utiliser les codes Last Z étape par étape

La méthode la plus sûre pour utiliser un code est de copier votre UID directement depuis le jeu et de coller le code exactement comme affiché. La plupart des échecs viennent de fautes de frappe, d’une casse incorrecte ou d’un UID incorrect.

  1. Ouvrez https://last-z.com/giftCenter/#/login
  2. Collez votre UID dans le champ UID
  3. Saisissez le Gift Code exactement comme affiché
  4. Touchez Confirm
  5. Ouvrez le jeu
  6. Vérifiez votre mailbox en jeu pour les récompenses

Notes importantes

  • L’UID est obligatoire — les codes ne fonctionneront pas sans lui
  • Les codes sont sensibles à la casse — copiez-collez pour éviter les erreurs
  • Les récompenses arrivent dans la mailbox en jeu, pas instantanément à l’écran
  • La livraison peut prendre quelques minutes — redémarrez le jeu si nécessaire
  • La plupart des codes sont à usage unique par compte — nouveau joueur ? Commencez avec notre guide débutant

Pourquoi votre code Last Z ne fonctionne pas

La plupart des échecs de redemption viennent de l’un de ces problèmes. Vérifiez-les dans cet ordre avant de supposer que le code est mort.

  1. Vous n’utilisez pas le Gift Center officiel. Les codes ne se saisissent pas dans le jeu.
  2. Votre UID est incorrect. Copiez-le directement depuis Settings au lieu de le taper à la main.
  3. Le code a été mal saisi. Utilisez copier-coller pour éviter les erreurs de casse.
  4. Le code a déjà été utilisé sur ce compte. La plupart des codes sont à usage unique par compte.
  5. Le code a expiré. Les codes d’événement et de palier disparaissent souvent rapidement.

Pour le dépannage complet étape par étape, utilisez la page Code Not Working.

Où sont mes récompenses de codes Last Z ?

Les récompenses sont livrées dans votre mailbox en jeu, elles ne s’affichent pas instantanément sur la page du Gift Center. Après redemption, ouvrez le jeu et vérifiez votre courrier. Si rien n’apparaît immédiatement, attendez quelques minutes et redémarrez le jeu une fois.
