Guide Second Laboratory

Research • Second Laboratory • Dernière revue mai 2026 • Last Z Guides

Le second Laboratory est un achat de vitesse Research, pas une exigence early game. Achetez-le après HQ18 seulement si vous pouvez garder deux files actives sans affamer le HQ, le Laboratory principal, les badges ou les branches combat clés.

Réponse rapide

Réponse courte : achetez le second Laboratory seulement s’il restera vraiment occupé. Il ajoute une file de Research, mais ne remplace pas la progression HQ, le Laboratory principal ni l’ordre Research de base.

Vérification avant achat

  1. Atteindre le point de déblocage HQ18.
  2. Confirmer que le Research License est disponible.
  3. Garder le Laboratory principal proche du niveau HQ.
  4. Acheter le second Laboratory seulement si la première file n’est jamais idle.
  5. Utiliser la deuxième file pour du Research utile pendant que la file principale reste focalisée.

Erreurs fréquentes

  • Achat trop tôt : si les ressources ou les badges sont déjà le goulot, une deuxième file ne règle pas le problème.
  • Mauvais lab prioritaire : le Laboratory principal reste le plus important pour les gros unlocks.
  • Research faible : une deuxième file accélère les choix, mais ne rend pas un mauvais choix efficace.

Étape suivante

Utilisez le Research Guide pour l’ordre principal, le Tech Guide pour la logique F2P, le Research Planner pour planifier les badges, et Field Research pour le scaling combat tardif.
