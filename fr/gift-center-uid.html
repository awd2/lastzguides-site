Configuration officielle du Gift Center Last Z : ouvrez le Gift Center dans un navigateur, copiez votre UID depuis Avatar → Settings → Copy ID, collez le code hors du jeu, puis récupérez les récompenses dans votre mailbox en jeu.

Connexion au Gift Center officiel

Si vous cherchez le Gift Center officiel de Last Z, utilisez la page de redemption ci-dessous. C'est le bon endroit pour reclamer les codes cadeaux et verifier le flux de base.

Gift Center officiel :
https://last-z.com/giftCenter/#/login

Besoin des codes actifs ? Utilisez la page Codes a reclamer. La configuration est correcte mais le code echoue encore ? Utilisez Code non fonctionnel ?.

Comment trouver et copier votre UID

Votre UID est obligatoire pour chaque redemption. La methode la plus fiable consiste a le copier directement dans le jeu afin d'eviter les erreurs de saisie.

  1. Ouvrez Last Z
  2. Touchez votre Avatar en haut a gauche
  3. Ouvrez Settings
  4. Touchez Copy ID
  5. Collez cet UID dans le Gift Center

Puis-je reclamer des codes Last Z sur iPhone ou Android ?

Oui. Le flux de redemption est le meme sur iPhone et Android, car les deux utilisent le site officiel du Gift Center dans un navigateur.

  • iPhone: ouvrez le Gift Center dans Safari ou un autre navigateur, puis collez votre UID et votre code.
  • Android: ouvrez le Gift Center dans Chrome ou un autre navigateur, puis collez votre UID et votre code.
  • Dans le jeu : non pris en charge pour la redemption de codes.

Peut-on reclamer les codes dans le jeu ?

Non. Les codes Last Z se reclament sur le site du Gift Center, pas dans le client du jeu. Si vous cherchez un menu de redemption dans l'application, vous regardez au mauvais endroit.

Where Do Recompenses Go After Redemption?

Recompenses are sent to your in-game mailbox. After redeeming, open the game and check your mail. If rewards do not appear instantly, wait a few minutes and restart the game once.

Erreurs frequentes de configuration du Gift Center

  • Utiliser le client du jeu au lieu du site web
  • Taper l'UID manuellement au lieu de le copier
  • Attendre les recompenses sur la page du Gift Center au lieu de verifier la mailbox
  • Oublier qu'iPhone et Android utilisent tous les deux un flux navigateur

Si votre configuration est correcte mais que le code echoue encore, utilisez la page de depannage Code non fonctionnel ?. Si vous avez d abord besoin de la liste actuelle des codes actifs, utilisez Codes a reclamer.

FAQ

What is the official Last Z Connexion au Gift Center page?

Use the official Connexion au Gift Center page: last-z.com/giftCenter/#/login.

Comment trouver mon UID dans Last Z ?

Ouvrez Avatar, allez dans Settings et touchez Copy ID. Votre UID est copie dans le presse-papiers et peut ensuite etre colle dans le Gift Center.

Puis-je reclamer des codes Last Z sur iPhone ou Android ?

Oui. Sur iPhone comme sur Android, la redemption des codes se fait via le site officiel du Gift Center dans votre navigateur.

Puis-je reclamer des codes dans le jeu ?

Non. Les codes Last Z se reclament via le site officiel du Gift Center, pas dans le client du jeu.

Ou vont les recompenses reclamees ?

Recompenses are sent to your in-game mailbox. Ouvrez le jeu apres la redemption et verifiez votre courrier.

À vérifier

  • L'élément important est le flux : Gift Center officiel, UID copié, redemption dans le navigateur, récompenses dans la mailbox.
  • Si la présentation du Gift Center change, gardez la même règle de sécurité : copiez votre UID depuis le jeu au lieu de le taper à la main.
  • Si les récompenses n'apparaissent pas tout de suite, vérifiez de nouveau la mailbox après quelques minutes avant de retenter le même code.
