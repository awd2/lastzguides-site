Connexion Gift Center et configuration UID Last Z — page officielle de codes

Guide support • Mis à jour en mars 2026 • Flux officiel du Gift Center • Last Z Guides

Réponse rapide

Configuration officielle du Gift Center Last Z : ouvrez le Gift Center dans un navigateur, copiez votre UID depuis Avatar → Settings → Copy ID, collez le code hors du jeu et récupérez les récompenses dans votre mailbox en jeu.

  1. ① Page officielle · Connexion au Gift Center
  2. ② Copier l’UID · Avatar → Settings → Copy ID
  3. ③ Flux mobile · navigateur sur iPhone ou Android
  4. ④ Récompenses · à récupérer dans la mailbox en jeu

🔐 Officiel uniquement : les codes sont utilisés sur le site du Gift Center, pas dans l’application du jeu.

🆔 Méthode UID la plus sûre : copiez votre ID directement depuis Settings au lieu de le saisir manuellement.

📬 Règle de la mailbox : les récompenses apparaissent dans votre courrier en jeu, pas sur l’écran du Gift Center.

Connexion officielle au Gift Center

Si vous cherchez le Gift Center officiel de Last Z, utilisez la page d’utilisation ci-dessous. C’est le bon endroit pour réclamer les codes cadeaux et vérifier le flux de base.

Gift Center officiel :
https://last-z.com/giftCenter/#/login

Besoin des codes actifs ? Utilisez la page Codes à utiliser. La configuration est correcte mais le code échoue toujours ? Utilisez Code non fonctionnel ?.

Comment trouver et copier votre UID

Votre UID est requis pour chaque utilisation de code. La méthode la plus propre consiste à le copier directement en jeu afin d’éviter les fautes de saisie.

  1. Ouvrez Last Z
  2. Touchez votre Avatar dans le coin supérieur gauche
  3. Ouvrez Settings
  4. Touchez Copy ID
  5. Collez cet UID dans le Gift Center

Puis-je utiliser les codes Last Z sur iPhone ou Android ?

Oui. Le flux d’utilisation est le même sur iPhone et Android, car les deux utilisent le site officiel du Gift Center dans un navigateur.

  • iPhone : ouvrez le Gift Center dans Safari ou un autre navigateur, puis collez votre UID et le code.
  • Android : ouvrez le Gift Center dans Chrome ou un autre navigateur, puis collez votre UID et le code.
  • En jeu : non pris en charge pour l’utilisation des codes.

Peut-on utiliser les codes dans le jeu ?

Non. Les codes Last Z sont utilisés via le site du Gift Center, pas dans le client du jeu. Si vous cherchez un menu de code dans l’application, vous cherchez au mauvais endroit.

Où vont les récompenses après utilisation ?

Les récompenses sont envoyées dans votre mailbox en jeu. Après l’utilisation du code, ouvrez le jeu et vérifiez vos messages. Si elles n’apparaissent pas immédiatement, attendez quelques minutes et redémarrez le jeu une fois.

Erreurs courantes de configuration du Gift Center

  • Utiliser le client du jeu au lieu du site web
  • Saisir l’UID manuellement au lieu de le copier
  • Attendre les récompenses sur la page du Gift Center au lieu de vérifier la mailbox
  • Oublier qu’iPhone et Android utilisent tous les deux un flux navigateur

Si votre configuration est correcte mais que le code échoue toujours, utilisez la page Code non fonctionnel ? de dépannage. Si vous avez d’abord besoin de la liste actuelle des codes actifs, utilisez Codes à utiliser.

FAQ

Quelle est la page officielle de connexion au Gift Center Last Z ?

Utilisez la page officielle de connexion au Gift Center : last-z.com/giftCenter/#/login.

Comment trouver mon UID dans Last Z ?

Ouvrez Avatar, allez dans Settings, puis touchez Copy ID. Votre UID est copié dans le presse-papiers et peut ensuite être collé dans le Gift Center.

Puis-je utiliser les codes Last Z sur iPhone ou Android ?

Oui. Sur iPhone comme sur Android, l’utilisation des codes se fait via le site officiel du Gift Center dans votre navigateur.

Puis-je utiliser les codes dans le jeu ?

Non. Les codes Last Z sont utilisés via le site officiel du Gift Center, pas dans le client du jeu.

Où vont les récompenses réclamées ?

Les récompenses sont envoyées dans votre mailbox en jeu. Ouvrez le jeu après l’utilisation du code et vérifiez vos messages.

À vérifier

  • Le bon flux reste le Gift Center dans un navigateur, pas le client du jeu.
  • L'UID doit être copié depuis Avatar → Settings → Copy ID, parce que le saisir à la main est l'erreur la plus facile.
  • Après une réclamation réussie, les récompenses doivent arriver dans ta boîte aux lettres en jeu.

Utilisez ce guide de configuration avant d’utiliser des codes Last Z. Copiez votre UID depuis le jeu au lieu de le saisir manuellement, passez par le Gift Center officiel et vérifiez votre mailbox après une soumission réussie.
